# 🚀 Trilha de Estágio | Compliance Soluções Fiscais

Bem-vindo ao repositório de acompanhamento da minha jornada de aprendizado como estagiário na **Compliance Soluções Fiscais**. Este projeto tem como objetivo documentar o progresso técnico, desafios superados e as entregas práticas realizadas durante a trilha de formação full stack baseada nos conteúdos da Rocketseat.

O foco principal desta trilha é o domínio do ecossistema JavaScript, utilizando tecnologias modernas como **Vue.js**, **Node.js** e **NestJS**.

## 📌 Objetivo do Programa
Capacitar o desenvolvimento de aplicações full stack completas, com domínio de frontend e backend, priorizando boas práticas de desenvolvimento e entendimento do fluxo completo de uma aplicação. O foco é construir uma base sólida, com clareza e evolução consistente.

---

## 🛠️ Stack Tecnológica
* **Frontend:** HTML, CSS e JavaScript.
* **Framework Frontend:** Vue.js 3.
* **Backend:** Node.js e NestJS.
* **Comunicação:** Conceitos de HTTP e consumo de APIs.

## 🚀 Estratégia de Execução
* **Consistência:** 1 a 2 horas de estudo por dia.
* **Prática:** Desenvolvimento de um projeto prático contínuo.
* **Revisão:** Acompanhamento com revisões semanais.
