# 🚀 Trilha de Formação Full Stack | Compliance Soluções Fiscais

Bem-vindo ao repositório de acompanhamento da minha jornada de aprendizado como estagiário na **Compliance Soluções Fiscais**. Este projeto tem como objetivo documentar o progresso técnico, desafios superados e as entregas práticas realizadas durante a trilha de formação full stack baseada nos conteúdos da Rocketseat [cite: 2].

O foco principal desta trilha é o domínio do ecossistema JavaScript, utilizando tecnologias modernas como **Vue.js**, **Node.js** e **NestJS** [cite: 2].

## 📌 Objetivo do Programa
Capacitar o desenvolvimento de aplicações full stack completas, com domínio de frontend e backend, priorizando boas práticas de desenvolvimento e entendimento do fluxo completo de uma aplicação [cite: 4]. O foco é construir uma base sólida, com clareza e evolução consistente [cite: 5].

---

## 🛠️ Stack Tecnológica
* **Frontend:** HTML, CSS e JavaScript.
* **Framework Frontend:** Vue.js 3.
* **Backend:** Node.js e NestJS.
* **Comunicação:** Conceitos de HTTP e consumo de APIs.
